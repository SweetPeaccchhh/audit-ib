# audit-ib
## О чем репозиторий?
Этот репозиторий создан для систематизации знаний о российском законодательстве в сфере информационной безопасности. Здесь я собираю ключевые нормативные акты, методики и мои конспекты.

## Мои аналитические заметки
*   Презентация: Анализ методики моделирования угроз ФСТЭК России от 5 февраля 2021 г. Обзор мирового опыта. Какие существуют базы данных угроз, атак, уязвимостей (https://docs.google.com/presentation/d/1ttP2H-35AIA0L1-L8-7iwsGxgBKbIeP_i29QIYhape0/edit?usp=sharing)
